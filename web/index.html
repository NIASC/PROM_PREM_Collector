PROM/PREM Collector application

This is the start page for the PROM/PREM Collector. The web version is under development, but the desktop application is available for download at the NIASC GitHub page.

The PROM/PREM Collector application can be started using Java Web Start (recommended) or as an applet (beta). The web start may require additional plugins such as IcedTea if using OpenJDK.
